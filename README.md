# ws-starter-theme
The "We studio - Starter Theme" is a custom starter theme for WordPress. Built by <a href="https://we-studio.ch/" title="We studio" target="_blank">We studio</a> in 2017.

## Support
- Responsive
- Flexible-width
- Multilingual
- Bootstrap 4
- Advanced Custom Fields
- WPML

## Build with
- CodeKit
