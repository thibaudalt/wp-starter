# wt-starter-theme
The "Web Tailor - Starter Theme" is a custom starter theme for WordPress. Built by <a href="http://web-tailor.ch" title="Web Tailor" target="_blank">Web Tailor</a> in 2019.

## Support
- Responsive
- Flexible-width
- Multilingual
- Bootstrap 4
- Advanced Custom Fields
- WPML
